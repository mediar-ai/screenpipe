## 图示概述

![screenpipe 图示](https://raw.githubusercontent.com/mediar-ai/screenpipe/main/content/diagram2.png)

1. **输入**: 屏幕和音频数据
2. **处理**: OCR、STT、转录、多模态集成
3. **存储**: SQLite 数据库
4. **插件**: 自定义管道
5. **集成**: Ollama、Deepgram、Notion、WhatsApp 等

这种模块化架构使 screenpipe 能够适应各种用例，从个人生产力跟踪到高级商业智能。
